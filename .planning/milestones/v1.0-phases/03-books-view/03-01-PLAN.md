Implementation requirements (all locked decisions from CONTEXT.md):

1. **Import only from existing modules** — use .js extensions (no bundler):
   - `import { fetchTable } from '../api.js';`
   - `import { renderError, renderEmpty } from '../components.js';`

2. **`renderBooks()` must be exported as `async function`** — the existing export signature changes from sync to async; router.js calls it without await so the sync skeleton render before any await is critical.

3. **Execution order inside renderBooks()**:
   - Line 1 (sync): `app.innerHTML = renderBooksSkeleton()` — layout visible immediately
   - Then: `const { data, error } = await fetchTable('books', { order: 'title' })`
   - Then: branch on error → renderError, empty → renderEmpty, else → renderBooksTable

4. **renderBooksTable(books)** — module-private function (not exported):
   - Wraps output in `<div class="max-w-4xl overflow-x-auto">`
   - `<h1 class="text-2xl font-bold text-white mb-6">Books</h1>`
   - `<table class="w-full text-sm text-left">`
   - Sticky header: `<thead class="sticky top-0 bg-gray-800 text-gray-400 uppercase text-xs">`
   - Column headers in order: Title | ASIN | Genre | Target Audience
   - `<tbody class="text-gray-100">` with one `<tr>` per book
   - Each `<tr>`: `class="border-b border-gray-700 hover:bg-gray-800 transition-colors"`
   - Title `<td>`: `class="py-3 px-4 max-w-xs truncate"` with `title="${escapeHtml(book.title)}"` tooltip attribute
   - ASIN `<td>`: contains `<a href="https://www.amazon.com/dp/${escapeHtml(book.kdp_asin ?? '')}" target="_blank" rel="noopener" class="text-indigo-400 hover:underline">${escapeHtml(book.kdp_asin ?? '—')}</a>`
   - Genre `<td>`: `class="py-3 px-4 text-gray-300"` — value: `escapeHtml(book.genre ?? '—')`
   - Target Audience `<td>`: `class="py-3 px-4 text-gray-300"` — value: `escapeHtml(book.target_audience ?? '—')`

5. **renderBooksSkeleton()** — module-private function (not exported):
   - Same outer wrapper and sticky header structure as renderBooksTable (prevents layout shift)
   - `<tbody>` contains exactly 4 skeleton rows (no class on tbody)
   - Each skeleton `<tr>`: `class="border-b border-gray-700"`
   - Each `<td>` contains `<div class="h-4 bg-gray-700 rounded animate-pulse {width}"></div>` with these widths per column:
     - Title: w-40
     - ASIN: w-24
     - Genre: w-20
     - Target Audience: w-32

6. **escapeHtml(str)** — module-private function (not exported):
   - Signature: `function escapeHtml(str)`
   - Body: `return String(str ?? '').replace(/&/g, '&amp;').replace(/</g, '&lt;').replace(/>/g, '&gt;').replace(/"/g, '&quot;');`

7. **Empty state**: call `renderEmpty('No books in your catalog yet')` — exact copy string, no variation

8. **Error state**: call `renderError(error)` — pass the error value directly from fetchTable destructure

Use Tailwind classes exclusively. No inline styles. No new dependencies. No other files modified.
