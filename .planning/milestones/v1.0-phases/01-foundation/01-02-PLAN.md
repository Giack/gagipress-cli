1. Create `vercel.json` at the repo root (not inside `dashboard/`):
```json
{
  "outputDirectory": "dashboard"

```
This tells Vercel to serve the `dashboard/` folder as the static site root, ignoring the Go source files.

2. Create `dashboard/index.html` using this structure:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Gagipress Dashboard</title>
  <!-- Tailwind CSS v4 CDN — must load before body content -->
  <script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>
  <!-- FOUC mitigation: hide body until Tailwind JIT compiles classes -->
  <style>body { opacity: 0; transition: opacity 0.15s; }</style>
  <script>
    document.addEventListener('DOMContentLoaded', function () {
      requestAnimationFrame(function () {
        document.body.style.opacity = '1';
      });
    });
  </script>
</head>
<body class="bg-gray-900 text-gray-100 min-h-screen">
  <nav class="flex gap-6 px-6 py-4 border-b border-gray-700 bg-gray-800">
    <a href="#/books"    class="text-gray-400 hover:text-white font-medium transition-colors">Books</a>
    <a href="#/ideas"    class="text-gray-400 hover:text-white font-medium transition-colors">Ideas</a>
    <a href="#/calendar" class="text-gray-400 hover:text-white font-medium transition-colors">Calendar</a>
  </nav>
  <main id="app" class="p-6">
    <!-- Views render here -->
  </main>
  <script type="module" src="js/app.js"></script>
</body>
</html>
```

Key requirements:
- The Tailwind CDN `<script>` tag must be in `<head>` before body content
- The FOUC mitigation inline `<style>` and `<script>` must come immediately after the Tailwind script tag
- `<main id="app">` is the mount point for all view content
- The entry point `<script type="module" src="js/app.js">` must use `type="module"` — no `defer` needed (modules are deferred by default)
- Do NOT use `class="dark"` on `<html>` — Tailwind v4 CDN does not use `.dark` class for dark mode utilities the same way; dark palette is applied via explicit `bg-gray-900` class on body instead

**`dashboard/js/router.js`:**
```javascript
import { renderBooks }   from './views/books.js';
import { renderIdeas }   from './views/ideas.js';
import { renderCalendar } from './views/calendar.js';

const routes = {
  '#/books':    renderBooks,
  '#/ideas':    renderIdeas,
  '#/calendar': renderCalendar,
};

export function dispatch() {
  const hash = location.hash || '#/ideas';
  const view = routes[hash] ?? renderIdeas;
  view();
}

window.addEventListener('hashchange', dispatch);
window.addEventListener('load', dispatch);
```
